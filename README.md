# Сортировка писем IT-поддержки
Проект разработывает систему, которая автоматически читает, классифицирует и раскладывает входящие письма по папкам.

## Перейдем к описанию работы программы.

Сначала программа считывает письма из папки data/inbox, которые получили из корпоративного почтового ящика, состоящего из ~100 писем.
Далее проект определяет тему, отправителя и основной текст письма. Анализируем ключевые слова и распределем письма на категории. 
В конце программа создаёт папки в output и сохраняет письма по соответствующим папкам.

## Категории

- Фишинг или спам
- Критический инцидент
- Оборудование
- Запрос доступа
- Проблема с ПО после обновления
- Мониторинг или оповещение
- Работа с клиентами
- Корпоративная рассылка
- Встреча или созвон
- HR или не IT
- Прочее

## Дополнительные фишки нашей программы
Мы реализовали статистику, которая считает количество писем и долю каждой категории.
Также в выводе программы добавился пьедистал, где рассположились 3 самые популярные категории.
